## 程序的机器级表示

- 一个可执行程序
    * 编译过程
        + 预处理
        + 编译
        + 汇编
        + 连接
    * 编译细节
        + `gcc -Og[1|2] <file>`优化等级
        + 编译器优化可能会调整代码次序，源代码会严重变形
- 数据格式
    * 字节，字，双字，四字
    * 数据类型的汇编代码表示
        + b, w, l, q 
- 操作数指示符
    * 操作数可以是立即数值、寄存器值和来自内存的值
    * 立即数的形式`$Imm`
    * 寄存器的形式`%r`
    * 内存寻址的形式`Imm(rb, ri, si)`
        + 表示`M[Imm + R[rb] + R[ri]*si`，参数是可以省略的，如`(,%rcx,)`
- 数据传送指令
    * `mov[数据大小]`
